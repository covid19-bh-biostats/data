# Datasets COVID19 evolution in Spain at national level

Data from daily Spanish Government reports. 
Data extraction and cleaning by datadista (it is also possible to have it by region). 

- *nacional_covid19.csv*: daily information about cases, hospitalisations, ICU and deaths in Spain
- *nacional_covid19_rango_edad.csv*: daily information about cases, hospitalisations, ICU and deaths in Spain by AGE and SEX group (not all reported cases, information about analysed cases in nota_nacional_covid_19_rango_edad.csv)

- *rename.R*: Small script to translate variables to english
- *nota_nacional_covid_19_rango_edad.csv*: Notes about data used in each day analysis.

source: Spanish Government Reports, extraction by Datadista https://github.com/datadista/datasets
